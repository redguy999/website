HTML
Hypertext Markup Language.
HTML element
A part of a HTML file.
HTML tag
Text is put between the start and end tags, said tags change how the text is displayed.
angle bracket
The less than and greater than symbols, used to create tags.
HTML attribute
Used to modify text within a tag beyond what the tag normally does
HTML comment
Text that is unviewable in the file, used to add notes for the dev(s).
HTML object element
HTML elements that allow HTML to do more than be a more intersting text file.
HTML special character
Characters that are either not on a normal keyboard or are normally used to make HTML tags.
